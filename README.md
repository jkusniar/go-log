go-log
===
[![GoDoc](https://godoc.org/github.com/jkusniar/go-log?status.svg)](https://godoc.org/github.com/jkusniar/go-log)

Package log is another implementation of logger in golang.

Features:

* simple,
* thread-safe, achieved using channels,
* supports log levels.
